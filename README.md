### 1. Calculator
A simple calculator that performs basic arithmetic operations.

**Features**

- Division

### 2. Contact Manager
A contact management system for storing and managing contacts.

**Features**
- Add contacts
- Search contacts
- Update contacts
- Delete contacts

### 3. Unit Converter
A converter for different measurement units.

**Features**
- Length conversion
- Weight conversion
- Temperature conversion

### 4. Dice Roller
A simple dice rolling simulator.

**Features**

- Multiple rolls

### 5. Password Generator

## Getting Started

### Prerequisites

- Python 3.x

### Run a Project

Clone the repository:

```bash
git clone https://github.com/Iamnikhilesh/python-project-collection.git
```

Navigate to the project folder:

```bash
cd python-project-collection
```

Run any project:

```bash
python Calculator.py
```
