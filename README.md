## Projects Included

### 1. Calculator
A simple calculator that performs basic arithmetic operations.

**Features**
- Addition
- Subtraction
- Multiplication
- Division

### 2. Contact Manager
A contact management system for storing and managing contacts.

**Features**
- Add contacts
- Search contacts
- Update contacts
- Delete contacts

### 3. Unit Converter
A converter for different measurement units.

**Features**
- Length conversion
- Weight conversion
- Temperature conversion

### 4. Dice Roller
A simple dice rolling simulator.

**Features**
- Random dice generation
- Multiple rolls

### 5. Password Generator
Generates secure random passwords.

**Features**
- Custom password length
- Random character generation

### 6. Quiz Game
A command-line quiz application.

**Features**
- Multiple questions
- Score calculation
- Result display

### 7. Rock Paper Scissors
Classic Rock-Paper-Scissors game against the computer.

**Features**
- Random computer choice
- Score tracking
- Winner determination

### 8. To-Do List
A simple task management application.

**Features**
- Add tasks
- Remove tasks
- View task list

## Getting Started

### Prerequisites

- Python 3.x

### Run a Project

Clone the repository:

```bash
git clone https://github.com/Iamnikhilesh/python-project-collection.git
```

Navigate to the project folder:

```bash
cd python-project-collection
```

Run any project:

```bash
python Calculator.py
```
